# Project Summary

## Overview

BuildTools Store is a web application for managing and presenting a live catalog
of construction equipment, industrial tools and job-site inventory.

## Frontend

- React-based interface for catalog browsing
- Search and category filters
- Product cards with pricing, stock and rating
- Product editor for creating and updating items
- Sass styling with reusable tokens and component-level structure

## Backend

- Express REST API for product management
- CRUD operations for catalog items
- Validation for text, price, stock and rating fields
- Seeded inventory for demo and local testing

## API Tooling

- Swagger UI for interactive endpoint documentation
- Postman collections for local API and external catalog API exploration

## Main Directories

- `frontend/`
- `backend/`
- `postman/`
- `reports/`
